# vue-ecology安装和初始化项目
```bash
## Project setup
yarn install

### Compiles and hot-reloads for development
yarn run serve

### Compiles and minifies for production
yarn run build

### Run your tests
yarn run test

### Lints and fixes files
yarn run lint
```
